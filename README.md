# **Project: Etch a Sketch**

In this project, an Etch-A-Sketch browser version has built.


### Demo: [Link](https://ev0clu.github.io/etch-a-sketch/)


## Features
- Dynamic grid selection and grid table update between 1x1 and 64x64 squares
- Color picker
- RGB color selection
- Erase function
